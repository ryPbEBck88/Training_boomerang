# PvP Mode

## Запуск без Redis (локальная разработка)

```powershell
$env:CHANNEL_LAYER='memory'
python manage.py runserver 8000
```

## Запуск с Redis

1. Установите и запустите Redis
2. `python manage.py runserver 8000`

## Использование

1. Зарегистрируйтесь или войдите
2. Перейдите в PvP
3. Выберите игру (например, AR: Соседи)
4. «Создать» — создать комнату и ждать соперника
5. «Найти» — найти ожидающую комнату или создать новую
6. Когда оба игрока в комнате, генерируется задание, оба вводят ответ
