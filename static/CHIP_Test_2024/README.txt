Положите сюда изображение текстуры для цилиндра и назовите chip.png
(или измените в шаблоне ar_bets.html имя файла в CYLINDER_TEXTURE_URL)

Как поправить текстуру:

1. ФАЙЛ chip.png:
   - Квадратное изображение (512x512 или 1024x1024)
   - Рисунок фишки строго по центру, до краёв (без белых полей)
   - Для круглой текстуры: круг в центре квадрата, углы можно залить цветом обода

2. В КОДЕ (ar_bets.html, блок с tex.repeat и tex.offset):
   - tex.repeat.set(0.9, 0.9) — уменьшить, если белая полоса по краям
   - tex.offset.set(0.05, 0.05) — сместить, если текстура съехала
   - tex.wrapS = tex.wrapT = THREE.ClampToEdgeWrapping — если Repeat даёт артефакты
